Patches?

One of the beauties of open-source is that you have the opportunity to fix bugs yourself instead of just reporting them. If you should choose to do so, please feel welcome to submit your patch to the main Simple-JNDI codebase. The correct way to generate a subversion patch is with the following command: 

svn diff src/java/org/osjava/sj/SimpleContext.java > SimpleContext-20050810.patch
However Simple-JNDI will accept the patch in any form, we're not fussy yet. Patches should be submitted to one of the following:
  • The OSJava Issue Tracker
  • The osjava-users mailing list
  • Failing that, directly to Henri Yandell
They'll be very much appreciated.
