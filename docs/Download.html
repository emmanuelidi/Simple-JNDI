Download Latest Code

The next version of simple-jndi will be 0.12. To get the current version of this you need to check the code out from Subversion and build it yourself using Maven. More info on Subversion is available on the svn-usage page and Maven can be obtained from maven.apache.org.

Download Latest Stable

  • simple-jndi-0.11.4.1-src.tar.gz
  • simple-jndi-0.11.4.1-src.zip
  • simple-jndi-0.11.4.1.tar.gz
  • simple-jndi-0.11.4.1.zip

If you require them through usage of optional features, please ensure you have the necessary dependencies:

  • commons-pool 1.3
  • commons-dbcp 1.2.2
  • (0.9.1 and before): gj-xml 1.0
  • (0.9.1 and before): commons-convert CVS HEAD build
